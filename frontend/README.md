# Ethan Website
The website is located at https://ethansun.org

## Quickstart
```
# Build backend
cd backend && npm ci && npm run build
## Install frontend dependencies
npm install
npm run dev
```

## Tech Stack
- Vite
- TypeScript
- React
- shadcn
